## Description
FarkleCPP is a text-based dice game implemented in C++. Players take turns rolling dice, scoring points according to Farkle rules, and competing to reach the winning score first. This project demonstrates structured programming, algorithm design, and console-based game mechanics.

## Features
- Classic Farkle dice game rules  
- Turn-based gameplay for multiple players  
- Score tracking and win condition logic  
- Planned enhancement: special dice types (e.g., "Joker") that alter gameplay strategy  

## Skills Demonstrated
- Programming fundamentals in C++  
- Problem-solving and logical reasoning  
- Command prompt interaction and user input handling  
- Gameplay mechanics design
